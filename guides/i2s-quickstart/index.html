What you'll get

A step-by-step guide that takes you from blank project to working I2S audio pipeline on STM32F411CEU6 with CS5343 (ADC) and CS4334 (DAC). Every section is grounded in a real audio effects processor — the code in the ZIP actually ran on hardware.

What's covered

  • Hardware wiring — CS5343 and CS4334 pinout to STM32F411
  • CubeMX clock tree and I2S3 peripheral configuration
  • Double-buffer DMA setup for glitch-free audio
  • Audio engine architecture (audio_engine.h / .c)
  • Common pitfalls: MCLK routing, DMA aliasing, CMSIS sign convention
